# Kuberna Labs Frontend

Architecting the Agentic Web3 Enterprise.

## Vision
To become the definitive operating system for the agentic economy, where intelligent agents seamlessly interact with both Web2 and Web3 systems.

## Design System
- **Typography**: Space Grotesk (Headings), Inter (Body)
- **Aesthetics**: iOS 26-inspired, glassmorphism, depth, and dynamic interactions.
- **Primary Color**: Indigo (#6366F1)
- **Secondary Color**: Emerald (#10B981)

## Tech Stack
- Next.js 14
- Tailwind CSS
- Framer Motion
- Lucide React
- Wagmi / Viem (Web3)
